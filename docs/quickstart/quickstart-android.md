Please check the Mobile resources available in the [Mobile development section](/docs/mobile). In particular, check the documentation about the development libraries:

- **Mobile UI:** a Kotlin multiplatorm library that contains the UI components used by the DHIS2 Android Capture app. Check the [Getting started](/docs/mobile/mobile-ui/getting-started) of the docs.
- **Android SDK:** a library to interact with DHIS2 instances and obtain offline storage for your app. Check the [Getting Started](/docs/mobile/android-sdk/getting-started) section of the docs.
